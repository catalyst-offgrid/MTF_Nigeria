# MTF_Nigeria
MTF module charts and data cleaning for Nigeria

This repo contains charts and datasets procured from *Nigeria - Multi-Tier Framework (MTF) Survey* comissioned by the World Bank and was first published on December 10, 2020. The World Bank URL for the underlying data, code book, etc. can be found here: `https://datacatalog.worldbank.org/search/dataset/0037865`

Spreadsheets in the `MTF_Nigeria/data` folder have been created using the R files and raw data in `MTF_Nigeria/src/data_processing`. Data available via the World Bank under CC0 license.
